CDAP Angular UI
===============


Global dependencies:

* `npm install -g bower gulp`

Local dependencies:

* `cd ../ngui`
* `npm install && bower install`

then, each in their own tab:

* `gulp watch` (autobuild + livereload)
* `npm start` (http-server)
* `npm test` (run karma for unit tests)
* `open http://localhost:8080`
